Theme

● Primary - Main colors most widely used across all screens and components. ● Accent - Also known as the secondary color. Used for floating action buttons and interactive elements. ● Warn - Colors to convey error state. ● Foreground - Used for text and icons. ● Background - Colors used for element backgrounds.

Pre-built themes

● deeppurple-amber ● indigo-pink ● pink-bluegrey ● purple-green

Defining a custom theme

● mat-core() This includes all common styles that are used by multiple components. This should only be included once in your application. ● Defines theme

Overlay-based components

Theming only certain components

@include mat-core-theme($candy-app-theme); @include mat-button-theme($candy-app-theme); @include mat-checkbox-theme($candy-app-theme);

Generate color palettes

$custom-theme-primary: mat-palette($mat-light-blue); $custom-theme-accent: mat-palette($mat-orange, A200, A100, A400); $custom-theme-warn: mat-palette($mat-red);
